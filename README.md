# Canteen_Utensils_Collector_Machine
A vending machine inspired machine that encourage users to return the utensils to specific place to control the hygiene of the canteen and improve discipline of users. Everytime when utensils are placed in the input of the machine, the sensor will detect and feed the utensils into the machine and dispense a coin for the user as a reward.
